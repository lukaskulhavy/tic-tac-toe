# Adastra Brand Kit

## Step 1 — Copy this folder into your project

Place the `adastra-brand-kit/` folder in the root of your web project, next to your main HTML file.

## Step 2 — Open the project in Claude Code and paste this prompt

```
Read `adastra-brand-kit/CLAUDE.md` and apply the brand to this project.
```

That's it.
